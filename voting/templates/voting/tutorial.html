Tutorial

Please read the documentation below in order to use the app efficiently.

Getting started

If yiu don't have admin privilages you can only vote and then check your vote. You are not required to login.

Voting

In order to take part in the voting proccess you only need to know your personal user id. Click the Start Voting button, provide your user id and the candidate id you want to vote for. Note that both your user id and the candidate id is just a number, so any other character will not be accepted and you will receive an error. Finally, each user is able to vote ony once. If you try to vote twice you will also receive an error.

Vote checking

You can check your vote anytime. Just click the Check Vote button and provide your user id.

Administrators

If you are an administrator you are able to see the whole blockchain.
